# todo-list_gpt

---

## 소개

이 리포는 python의 flask와 sqlite 라이브러리를 사용하여 만든 간단한 todo list입니다.
또한, chat gpt의 기능을 이용하여 만들었으며 개인 공부용 자료입니다.

---

## 주요기능

 - todo list add
 - todo list delete
 - todo list check func.
 - simple ui

---

## 사용법

1. python에서 flask install
2. app.py 실행
3. 브라우저에서 http://127.0.0.1:5000 접속 혹은 터미널 로그에 나오는 주소로 접속
